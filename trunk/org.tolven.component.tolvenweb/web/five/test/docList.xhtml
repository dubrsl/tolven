A list of documents owned by any account of which the the logged in user is a member. This is a forced security violation for demo purposes only. Normally, a user is not allowed to see documents (or anything) in accounts other than the one they are logged into. This query shows that if that restriction were breached, documents in an account other than the account the user is currently logged into, even if that user is a member of the other account, will not be viewable or computable because the document is encryped.

In order to ssee the failure, you must have access to more than one account and attempt to read a document in an account you are not currently logged into.
