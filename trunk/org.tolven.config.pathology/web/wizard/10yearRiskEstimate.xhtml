The data for the 10 year risk estimate is generated from “Adjuvant on line” at (http://www.adjuvantonline.com/index.jsp). This Tolven wizard allows you to

1) Enter the essential numbers from the Adjuvant on Line application into Tolven so that you can use these metrics for more detailed queries and comparisons and you can track trends if the patient has subsequent 10 year risk calculations

2) Capture the actual page from the Adjuvant on Line site, enabling you to have an electronic copy of the information in the patient record. This step is optional.

In order to capture the image from Adjuvant on Line, when you have completed the Adjuvant on Line calculation, use the “Print Screen” function on your computer to save the image to a blank power point slide. Once this slide has been saved to the computer’s hard drive, you may follow the instructions on the wizard that will allow you to attach the power point file so that the primary data can be saved as well as the data that you have transcribed into the wizard.
